## Steps

<Steps>
  <Step title="Create main sections">
    ````
    Use the "Add Section" button to group related pages (e.g., Guides, FAQs).
    ````
  </Step>
  <Step title="Add pages">
    ````
    Click "New Page" to add articles, how-tos, or API references.
    Use Markdown or the editor.
    ````
  </Step>
  <Step title="Reorder content">
    ````
    Drag and drop sections or pages to arrange navigation.
    ````
  </Step>
  <Step title="Publish">
    ````
    Press "Publish" when ready to make docs public.
    ````
  </Step>
</Steps>
