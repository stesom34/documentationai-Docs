## Getting Started

To begin using Сергій Москалюк's documentation space:

1. Sign in at `https://sergii-moskaliuk.com`.
2. Click "Create New Project" from your dashboard.
3. Name your project and add a description.
4. Use the editor to add pages, FAQs, and guides.
5. Click "Publish" to make docs available.

Need more help?
- Visit the [Guide](/help-center/guides/organize-your-docs)
- Contact support if you get stuck.
