## Quick Start

Get up and running in minutes. Follow these steps to create your first documentation page.

<Steps>
  <Step title="Create a New Page" icon="file-plus">
    Navigate to your project root and add a new file like `introduction.mdx`.

    ```bash
    mkdir docs
    touch docs/introduction.mdx
    ```
  </Step>
  <Step title="Add Frontmatter" icon="settings">
    Start every page with YAML frontmatter. No blank lines inside!

````mdx
---
title: Your Page Title
description: Brief summary here.
---
````

  </Step>
  <Step title="Write Content" icon="edit-3">
    Use H2 headings and components. Here's a simple `<CodeGroup>` example:

    <CodeGroup tabs="JavaScript,Python">
    ````javascript
    console.log("Hello, Сергій Москалюк!");
    ````
    ````python
    print("Hello, Сергій Москалюк!")
    ````
    </CodeGroup>
  </Step>
  <Step title="Preview and Publish" icon="eye">
    Run your local server to preview changes, then deploy to production.
  </Step>
</Steps>
