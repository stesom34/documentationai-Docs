## Recent Updates

Stay informed about the latest releases for the Сергій Москалюк documentation platform. You receive new features, performance improvements, and bug fixes to enhance your documentation experience.

<Update label="2024-11-15" description="v2.0.0" tags={["breaking", "feature", "improvement"]}>

## Breaking Changes

- Removed legacy `{legacyConfig}` support in MDX parsers. Migrate to `{modernConfig}` for compatibility.

## New Features

- Added real-time collaboration editing with live cursors and conflict resolution.
- Introduced AI-powered content suggestions using natural language prompts.
- Support for custom brand colors, like `#3B82F6`, directly in theme configurations.

## Improvements

- Enhanced search performance by `{>300%}` for large documentation sites.
- Optimized image loading with lazy-loading and WebP format conversion.

## Bug Fixes

- Fixed parsing errors with nested code blocks using 4 backticks.
- Resolved anchor link issues in expandable sections.

</Update>

<Update label="2024-10-20" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Mermaid diagram rendering as secure code blocks, not JSX components.
- Added `<Update>` component for streamlined changelog management.

## Improvements

- Improved JSX attribute validation to prevent common parsing errors like unescaped `{` characters.

## Bug Fixes

- Corrected `ParamField` rendering when using `path` and `query` simultaneously.
- Fixed `Tabs` component icon alignment on mobile devices.

</Update>

<Update label="2024-09-10" description="v1.0.0" tags={["feature"]}>

## Initial Release

- Core MDX documentation engine with 16+ components including `Cards`, `Steps`, and `CodeGroup`.
- Full support for YAML frontmatter, H2-first heading structure, and special character escaping.
- Built-in syntax highlighting for `javascript`, `python`, `bash`, and more.

</Update>
